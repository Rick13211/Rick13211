## Featured Projects

<details open>
<summary><strong>Raahi — Safety-Aware Navigation for India</strong></summary>

<br/>

A navigation platform that evaluates not only how fast a route is, but how safe it may be.

| Area | Implementation |
|---|---|
| Stack | Next.js 15, TypeScript, Tailwind CSS, Zustand, Supabase, PostgreSQL, PostGIS |
| Geospatial | Mapbox GL JS, OSRM, Turf.js, OpenStreetMap, Overpass API |
| Core Engine | 8-factor weighted route-scoring system producing a 0–100 score |
| Integrations | Weather, accident data, street lights, community reports, route geometry and SMS |
| Features | Alternative route comparison, reports, SOS, analytics and nearby safe zones |
| Repository | [View Source](https://github.com/Rick13211/raahi_) |

Raahi combines street lighting, historical crime indicators, government accident data, community reports, safe-zone proximity, urban activity signals, time of day, and weather into an explainable route-level safety score.

</details>
